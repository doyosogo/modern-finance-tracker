# FinanceTracker

FinanceTracker is a personal finance management application built with React. The project allows users to track income and expenses, monitor budgets, create savings goals, analyse spending habits, and manage their financial information through a clean and simple interface.

This project was created as part of my software development portfolio to practise frontend development, data visualisation, state management, and working with browser storage.

## Features

- Add, edit, and delete transactions
- Track income and expenses
- Search and filter transaction history
- Create recurring transactions
- Set category budgets
- Monitor budget progress
- Create savings goals
- Financial insights and spending analysis
- Interactive charts and reports
- Export and import backups
- Local data persistence using Local Storage

## Technologies Used

- React
- Vite
- JavaScript (ES6+)
- Tailwind CSS
- Recharts
- Local Storage

## Installation

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/finance-tracker.git
```

Navigate into the project:

```bash
cd finance-tracker
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```
